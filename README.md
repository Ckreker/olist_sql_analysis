# Анализ данных бразильского маркетплейса Olist

## О проекте
Анализ данных интернет-магазина Olist с использованием PostgreSQL.

## Технологии
- PostgreSQL
- DBeaver
- SQL

## Структура проекта
- `sql/` — SQL-запросы для анализа
- `insights` — выводы по результатам

## Ключевые выводы
Olist — успешная платформа с высоким процентом доставленных заказов (97%). Главная проблема — низкая повторная покупаемость. Решение — программа лояльности, скидки на повторные заказы и улучшение логистики для крупных товаров. Сезонные спады (сентябрь-октябрь) можно компенсировать акциями.
